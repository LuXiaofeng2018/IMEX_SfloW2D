type(bc), dimension(n_vars) parameters_2d::bcd

blD&flag defines the bottom (down) boundary condition:

  • bcDflag = 0 => Dirichlet boundary condition;
  • bcDflag = 1 => Neumann boundary condition.

bcDvalue is the value of the bottom boundary condition:

  • value of the variable for Dirichlet boundary condition (bcDflag=0);
  • gradient of the variable for Neumann boundary condition (bcDflag=1).

integer, dimension(n_vars) parameters_2d::limiter

Limiter for the slope in the linear reconstruction:
.

  • 'none' => no limiter (constant value);
  • 'minmod' => minmod sloe;
  • 'superbee' => superbee limiter (Roe, 1985);
  • 'van_leer' => monotonized central-difference limiter (van Leer, 1977)
